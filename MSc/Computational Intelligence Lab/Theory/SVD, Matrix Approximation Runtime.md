## SVD Runtime
$\mathcal{O}(\min(nm^2, mn^2))$
See 2.3.4

Can be used for a (normal, fully-observed) low-rank approximation.
## Weighted low-rank approximation runtime
Weighted low-rank matrix approximation is NP-hard.

In particular: For matrix completion, weigh by observation (observed/missing).
